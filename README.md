### etch-a-sketch
A browser version of something between a sketchpad and an Etch-A-Sketch.

### How to Get Started
- Open the project: Just open index.html in your favorite web browser

- Start drawing: Move your mouse over the grid while holding down the mouse button

- Get creative: Try all the different modes and settings!

### Drawing Modes
- Rainbow Mode : Creates vibrant, randomly colored squares

- Black Mode : Draws in classic black (with optional shading effects)


Button |	What It Does |	Cool Tip!|
------------------|-----------------------------|----------------------------------------|
Shading: ON/OFF |	Toggles shading effects - squares get darker each time you draw over them |	Try shading with rainbow mode for a cool layered effect!|
------------------|------------------------------|-----------------------------------------|
New Grid |	Clears everything and creates a fresh grid |	Perfect for starting a new masterpiece!|
------------------|------------------------------|-----------------------------------------------|
Grid Size Select |	Choose how many squares your grid has (10-100) |	Smaller number = bigger squares, bigger number = finer detail!|
